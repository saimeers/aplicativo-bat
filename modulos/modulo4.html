LOGMAN

Es una herramienta de línea de comandos en Windows que se utiliza para crear y administrar sesiones de recolección de datos de rendimiento y eventos. Permite a los administradores y usuarios avanzados capturar y analizar datos sobre el rendimiento del sistema y aplicaciones.

Sintaxis

logman [create | query | start | stop | delete| update | import | export | /?] [options]

Parámetros

En esta sección se muestran los parámetros comunes utilizados con logman, junto con sus descripciones correspondientes.

    • create: Crea una nueva sesión de recolección de datos.
    • update: Actualiza la configuración de una sesión de recolección de datos existente.
    • query: Consulta la configuración y estado de las sesiones de recolección de datos.
      • Se pueden combinar los siguientes parametros con los anteriores:
      • -c: Especifica los contadores de rendimiento a recopilar.
      • -si: Especifica el intervalo de muestreo en segundos.
      • -o: Especifica el archivo de salida donde se guardarán los datos recopilados.
    • start: Inicia una sesión de recolección de datos existente.
    • stop: Detiene una sesión de recolección de datos en ejecución.
    • delete: Elimina una sesión de recolección de datos.
