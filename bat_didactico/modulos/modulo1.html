NETSH

Es una herramienta de línea de comandos en los sistemas operativos Windows que se utiliza para configurar y gestionar varios aspectos de la red y la configuración del sistema.

El nombre "Netsh" es una abreviatura de "Network Shell". Puedes utilizar "Netsh" para realizar una amplia variedad de tareas relacionadas con la configuración y el diagnóstico de la red en Windows.

Sintaxis

netsh [-a Aliasfile ] [-c Context ] [-r Remotecomputer ] [-u [ domainname \username ]] [-p Password | [{NetshCommand | -f scriptfile}]

Parámetros

En esta sección se muestran los parámetros comunes utilizados con netsh, junto con sus descripciones correspondientes.

  • interface
    • show interface: Muestra una lista de todas las interfaces de red en el sistema.
    • set interface: Configura el estado de una interfaz de red (activada o desactivada).
    • ip set address: Configura la dirección IP de una interfaz de red.
    • ipv4 set address: Configura la dirección IPv4 de una interfaz de red.
    • ipv4 set dnsservers: Configura los servidores DNS de una interfaz de red.
    • ipv4 add dnsservers: Añade servidores DNS adicionales a una interfaz de red.
  • wlan
    • show interfaces: Muestra la configuración de las interfaces WLAN (Wi-Fi).
    • show profiles: Muestra una lista de todos los perfiles WLAN en el sistema.
    • add profile: Añade un nuevo perfil WLAN desde un archivo XML.
    • delete profile: Elimina un perfil WLAN especificado.
